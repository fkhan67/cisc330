1) The Internet Service Provider (ISP) is contacted by the browser when a user puts a URL into their browser to obtain the associated IP address. The Domain Name System (DNS) is used by the ISP to translate the URL into an IP address. After then, a request is sent by the browser to the web server that its IP address indicates. After processing the request and obtaining the website, the web server sends it back to the browser so that it may be rendered and viewed by the user.
2)Hypertext Markup Language(HTML) consists of a group of elements denoted by tags that encompass several content parts, including headings, paragraphs, links, and images. HTML establishes the foundation for content creation and organization on the World Wide Web by enabling web browsers to understand and display material correctly.
3) The site has a distinct header as well as multiple links to different sections. Also includes images and other descriptive headings.
4)An HTML element is a fundamental building block of an HTML document. It consists of a start tag, content, and an end tag. An HTML tag is a set of characters enclosed in angle brackets that define the beginning and end of an element. An attribute provides additional information about HTML elements.
